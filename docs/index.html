¿Cómo usar el contructor de acordes?

  1. Selecciona la nota raíz del acorde (ej. C, D, E...)
  2. Agrega una alteración si la deseas (# o ♭)
  3. Elige el tipo de acorde (mayor, menor, 7, sus, etc.)

Al seleccionar, verás el nombre del acorde y las notas que lo componen. Luego puedes reproducirlo con el botón 🎵.
